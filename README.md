# Workshop: Farsi lernen

> **[🌐 Landing Page](https://open-learn.app/workshop-farsi/)** · **[🚀 Start Workshop](https://open-learn.app/#/add?source=https://open-learn.app/workshop-farsi)**

5 Lektionen zum Erlernen der wichtigsten Wörter und Ausdrücke auf Farsi (Persisch). Available in German, English, and Farsi interfaces.

## Content

### 5 Lessons
1. **Greetings** — Basic greetings and introductions
2. **Everyday Words** — Common daily vocabulary
3. **Numbers & Time** — Counting and time expressions
4. **Food & Shopping** — Food vocabulary and shopping phrases
5. **Travel & Directions** — Navigation and travel vocabulary

## Languages
- 🇩🇪 Deutsch → Farsi
- 🇬🇧 English → Farsi
- 🇮🇷 Farsi → Deutsch / English
